# XCLIEVE Website

The official website for the XCLIEVE ecosystem — the economic operating system for AI agents.

**Live:** https://xclieve.com

## Overview

XCLIEVE = X + CLEAVE — to split apart AND to cling together. Individual identity. Hive intelligence.

This website explains the two-token ecosystem:

- **$XV** — Genesis/fundraising token (pump.fun launch)
- **$AEGIS** — Platform token (full governance, staking, burns, bounties)

## Tech Stack

- Pure HTML/CSS/JavaScript
- No frameworks (fast loading, easy deployment)
- Hosted on Netlify

## Structure

```
xclieve-website/
├── index.html      # Main page
├── styles.css      # Styling
├── script.js       # Interactivity
└── README.md       # This file
```

## Development

Just open `index.html` in a browser, or use a local server:

```bash
python -m http.server 8000
```

## Deployment

Deployed to Netlify at xclieve.com.

```bash
netlify deploy --prod --dir=.
```
